# Ren'Py_automatic_translations

Translate your Ren'Py projects with [deep-translator](https://github.com/nidhaloff/deep-translator) (Google Translate backend).

## Requirements

1. Python 3
2. Install dependencies:

```bash
pip install deep-translator pycountry
```

(`requests` is pulled in by `deep-translator` and is used for retryable network errors in parallel mode.)

## Usage

From this repository’s root:

```bash
python src/auto_translate_gui.py
```

1. **Source TL folder** — Browse to the game’s **`…/<game name>/game/tl`** folder (e.g. under Steam). The tool reads strings from here. The **game name** is taken automatically as the parent of the `game` folder (the folder that contains `game/tl`).
2. Translated files are written under **this repo** at:

   **`RenPy_automatic_translations/<same game name>/game/tl/`**

   Example: if you select `…\Come Home\game\tl`, output goes to `…\RenPy_automatic_translations\Come Home\game\tl\…`.

3. Choose the **language folder** to translate and your **source ISO** code.
4. **Translate** — full run; **Resume** — missing/empty outputs or still-unfilled `""` lines under that output tree.
5. **Workers** — parallel worker count; request pacing is **automatic** (see **Rate limiting** below).

Session logs are written to **`RenPy_automatic_translations/logs/auto_translate_gui_YYYYMMDD_HHMMSS.log`**.

### Rate limiting

All workers share an **adaptive** pacing limiter: it starts with no extra delay, **increases** spacing when HTTP **429** / “too many requests” / “rate limit” style errors occur, and **relaxes** gradually after successful translations.

Each run ends with:

- `Translated: X / T`
- `Failed: Y / T`

## Output safety

Each `.rpy` is written to a temporary file next to the final path, then moved into place with `os.replace`, so a failed translation does not leave a partial `.rpy` in the output tree.

## Notes

- Language folder names under `tl` must match [pycountry](https://pypi.org/project/pycountry/) English names (lowercase), as generated by the Ren'Py launcher.
- List of ISO 639-1 codes: [Wikipedia — ISO 639-1](https://en.wikipedia.org/wiki/List_of_ISO_639-1_codes)
